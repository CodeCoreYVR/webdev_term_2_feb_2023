[Lab] Temp Converter

Build a Node HTTP server:
- It will take a temperature in Fahrenheit from the url params. (e.g. given "http://localhost:8080/?temp=100" takes 100)
- It will convert that temperature to Celsius and display it in browser.


------------------- To Run -------------------
- Open terminal window
$ node server.js
- Open browser tab or window and type into the address bar: 
    http://localhost:3000/?temp=100
